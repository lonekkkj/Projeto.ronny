# Ronny
Ensinando código, conectando talentos e transformando o futuro.

## **Sobre o Projeto**

O Projeto Ronny é uma plataforma educacional inclusiva desenvolvida com o objetivo de democratizar o acesso ao ensino de tecnologia para estudantes neurodivergentes, especialmente jovens com Transtorno do Espectro Autista (TEA). A proposta surgiu a partir da necessidade de tornar o aprendizado de programação mais acessível, reduzindo barreiras cognitivas e sensoriais presentes nos métodos tradicionais de ensino.

Atualmente, muitos estudantes com TEA enfrentam dificuldades para acompanhar disciplinas relacionadas à lógica de programação e pensamento computacional devido ao excesso de estímulos visuais, interfaces complexas e metodologias abstratas utilizadas pelas plataformas convencionais. Essa realidade contribui para a desmotivação, ansiedade, evasão escolar e exclusão digital de milhares de jovens neurodivergentes.

Diante desse cenário, o Projeto Ronny foi idealizado como uma solução inovadora que combina acessibilidade cognitiva, tecnologia educacional e design minimalista para oferecer uma experiência de aprendizagem mais confortável, previsível e inclusiva.


## **Problema Abordado**


Embora a inclusão escolar tenha avançado significativamente nos últimos anos, estudantes neurodivergentes ainda encontram inúmeras dificuldades no acesso ao ensino tecnológico. A maior parte das plataformas de programação existentes foi desenvolvida para usuários neurotípicos, apresentando excesso de cores, sons, animações e informações simultâneas que acabam causando sobrecarga sensorial em alunos com TEA.

Além das dificuldades visuais e cognitivas, o ensino tradicional de programação geralmente exige contato imediato com sintaxes complexas e códigos textuais abstratos. Para muitos estudantes autistas, isso gera ansiedade, bloqueios de aprendizagem e frustração constante diante dos erros encontrados durante as atividades.

Outro problema identificado é a dificuldade enfrentada pelos professores do Atendimento Educacional Especializado (AEE), que frequentemente precisam adaptar conteúdos manualmente para atender às necessidades específicas de cada aluno. Esse processo exige tempo, esforço e conhecimento técnico, tornando a inclusão mais difícil de ser aplicada em larga escala.

O Projeto Ronny surge justamente para enfrentar essas barreiras, transformando o ensino de lógica e programação em uma experiência visual, intuitiva e sensorialmente amigável.


## **Solução Desenvolvida**


A solução proposta consiste em uma plataforma educacional web/mobile baseada em Arquitetura de Acessibilidade Cognitiva. O sistema foi desenvolvido para traduzir conceitos abstratos da programação em experiências práticas e visuais, utilizando blocos lógicos interativos e mecânicas drag-and-drop.

Ao invés de exigir que o aluno escreva códigos complexos desde o início, a plataforma permite que ele aprenda conceitos fundamentais de pensamento computacional por meio de desafios gamificados organizados de forma simples e previsível. Essa abordagem reduz significativamente a carga cognitiva e facilita a compreensão lógica.

A interface foi construída utilizando princípios de Design Universal para Aprendizagem (DUA), priorizando ambientes de baixo estímulo visual, dark mode nativo, organização limpa das informações e feedbacks suaves, sem punições agressivas ou alertas sonoros excessivos.

Enquanto o estudante utiliza a plataforma, o sistema coleta dados pedagógicos de forma silenciosa e gera relatórios inteligentes para os educadores. Dessa forma, professores conseguem acompanhar dificuldades específicas, analisar desempenho e construir Planos de Desenvolvimento Individual (PDI) de maneira mais eficiente.

Mais do que ensinar programação, o Ronny busca promover autonomia, inclusão digital, segurança emocional e preparação profissional para jovens neurodivergentes.


## **Tecnologias Utilizadas**


O desenvolvimento do Projeto Ronny foi estruturado utilizando tecnologias modernas e acessíveis, priorizando agilidade, escalabilidade e facilidade de manutenção. Durante a etapa de prototipagem e validação visual, foram utilizadas ferramentas como Figma e metodologias de Design Thinking para construção da arquitetura cognitiva da plataforma.

A construção do MVP foi planejada utilizando soluções Low-Code, como FlutterFlow, permitindo o desenvolvimento rápido de uma aplicação web/mobile responsiva e multiplataforma. Para armazenamento de dados, autenticação e gerenciamento de informações em tempo real, foi adotado o Firebase como infraestrutura principal do sistema.

Além das tecnologias de desenvolvimento, o projeto também se fundamenta em metodologias como DMAIC, Disciplined Entrepreneurship e Design Universal para Aprendizagem, garantindo que a plataforma mantenha foco em melhoria contínua, acessibilidade e validação pedagógica.

A página web do projeto Ronny foi desenvolvida com:

- HTML5
- CSS3
- JavaScript moderno (ES6+)
- Blockly
- Vite
- Bun
- APIs de acessibilidade do navegador

Tudo isso sendo programado no Visual Studio Code, utilizando uma arquitetura Front-End moderna.

## **Objetivos do Projeto**


O principal objetivo do Projeto Ronny é garantir acesso equitativo ao ensino de pensamento computacional para estudantes neurodivergentes. A plataforma busca reduzir barreiras cognitivas, minimizar sobrecarga sensorial e transformar o aprendizado de programação em uma experiência acessível e acolhedora.

Além disso, o projeto pretende auxiliar professores na adaptação curricular, reduzir índices de evasão escolar em disciplinas tecnológicas e desenvolver competências valorizadas pelo mercado de tecnologia, como lógica, resolução de problemas e reconhecimento de padrões.

A longo prazo, o Ronny também possui potencial de expansão para atender outras neurodivergências, como TDAH e Dislexia, ampliando ainda mais seu impacto social e educacional.


## **Instruções de Execução**


Para executar o projeto localmente, é necessário possuir um ambiente de desenvolvimento configurado com as ferramentas utilizadas pela plataforma. Inicialmente, o usuário deve clonar o repositório do GitHub utilizando o comando de clonagem do Git.

Após acessar a pasta do projeto, as dependências devem ser instaladas utilizando o gerenciador de pacotes do Flutter. Em seguida, a aplicação pode ser executada localmente através do comando de inicialização da plataforma.

O sistema foi desenvolvido para funcionar tanto em ambiente web quanto mobile, garantindo flexibilidade e acessibilidade para estudantes e educadores.


## **Impacto Esperado**


O Projeto Ronny possui como principal propósito promover inclusão tecnológica e transformação social através da educação. A plataforma busca diminuir barreiras históricas enfrentadas por estudantes neurodivergentes no acesso à tecnologia, oferecendo um ambiente mais seguro, acolhedor e adaptado às suas necessidades cognitivas.

Espera-se que a solução contribua para a redução da evasão escolar, aumento do engajamento nas disciplinas tecnológicas e fortalecimento da autonomia dos estudantes. Além disso, o projeto também visa preparar jovens neurodivergentes para futuras oportunidades profissionais no setor de tecnologia da informação.

Ao unir acessibilidade cognitiva, inovação pedagógica e tecnologia educacional, o Ronny se consolida como uma iniciativa capaz de transformar a relação entre inclusão e ensino de programação no contexto educacional brasileiro.
